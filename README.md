# HTML_CSS_Test

Installation local environment
------------

1. Download and unzip the project
2. NodeJS installed(http://nodejs.org/)
3. GulpJS installed globally
```
npm install -g gulp
```
4.
```
npm install gulp gulp-sass browser-sync -save-dev
```
5.
```
gulp watch
```
